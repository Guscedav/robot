Detailed Description

This class represents a 4x4 homogeneous transformation matrix used to describe the pose of a coordinate system. It offers methods to multiply itself with another transformation, to rotate the represented coordinate system around the local X, Y or Z axis, to translate the coordinate system in X, Y and Z direction, and a number of other miscellaneous methods.
