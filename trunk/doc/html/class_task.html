Increments the robots motion states in all coordinate spaces.

This method is called by the task sequencer periodically. It increments the states of the cartesian pose, the joint angles and the actuator coordinates for a given time period.

Parameters
posethe robots pose in Cartesian coordinates. This transformation needs to be incremented by this method.
velocitythe robots velocity in Cartesian coordinates.
accelerationthe robots acceleration in Cartesian coordinates.
phian array of motion states of the joint angles. These motion states need to be incremented by this method.
qan array of motion states of the actuator coordinates. These motion states need to be incremented by this method.
velocityOverridean override parameter to scale the profile velocity with.
periodthe time period to increment the motion states for, given in [s].
Returns
the status of this task, i.e. FAULT, RUNNING or DONE.

Reimplemented in TaskWaitDI, TaskWait, TaskStop, TaskSetDO, TaskMoveToolWithJoystick, TaskMoveLinear, and TaskMoveJoint.
