Detailed Description

This is a task that moves the robot to a given Cartesian pose.

Creates a task object that moves the robot to a given Cartesian pose.

Parameters
robotModela reference to the robot model object to use.
toolthe given tool to use for the calculation of the direct kinematic equations.
posea transformation matrix defining the desired Cartesian position and orientation of the robots tool.
configurationa parameter defining the desired configuration of the robot.
velocitythe velocity at which to move the robot, given in [m/s].
zonethe minimum distance to the desired pose before the robot moves to the next task in the task list, given in [m]. A zone parameter of 0.0 means that the robot moves precisely to the given pose and stops, before moving to the next task in the task list.
