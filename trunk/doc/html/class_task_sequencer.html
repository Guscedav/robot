Detailed Description

The task sequencer processes a list of tasks, like tasks to move the robot to a given pose, tasks to control a digital output (i.e. to control a gripper), tasks to wait for a specified duration, etc.
This class offers a number of methods to set tasks to process and to check the state of the task sequencer. And since it is a realtime thread doing the processing, it also offers methods to set this thread's priority and period, and methods to start and stop this thread.
